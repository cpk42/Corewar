---------------DO NOT PUSH WORK TO MASTER UNLESS TOLD OTHERWISE----------------

Master branch is only to be touched when development branch is ready to update.

###############################################################################

To create a branch off of the development branch:

- git branch -- (list all current branches in repo)

- git checkout development -- (switch to development branch)

- git checkout -b intra_branch -- (create branch off of development)

You should now be on a new branch created from the latest version of Development

################################################################################

When you believe your work is ready to be merged with Development, push to your 
branch and create pull request. Message Curtis before any commits are made to
master OR development.
